## Quick Start

1. Upload a .bam or .fasta file to Latch Data
   ![](/images/latch-data.png)
2. Double click that file to open it
3. IGV will automatically spin up a session

## Inputs

Any .bam, .sam, and .fasta

## Outputs

An IGV session will spin up automatically like this

![](/images/igv-2.png)
